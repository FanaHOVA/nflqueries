About NFL Queries

NFL Queries lets you search for NFL facts through everyday language rather than boring dropdown menus.

It started out as an experiment in Javascript that I ended up working on further. I've written a bit about the tech behind this on my blog if you're interested.

At the moment you can search for weight, jersey number, college and number of years in the NFL. Active players can be filtered by team and there's also an injured option so that you can easily lookup injured players on each team. The database has been created with the Python library nflgame and that's what I'm gonna use to keep it updated, so big shoutout to the creator Andrew Gallant!
